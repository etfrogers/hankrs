# hankrs

`hankrs` is a Rust implementation of the quasi-discrete Hankel transform. It was designed for use primarily in cases where a discrete Hankel transform is required, similar to the FFT for a Fourier transform.

## Example Gallery

This documentation hosts a gallery of usage examples demonstrating the capabilities, speed, and accuracy of `hankrs`.

Use the sidebar to navigate through the gallery and see `hankrs` in action. All plots and outputs shown in the examples are generated directly by the underlying Rust code.
